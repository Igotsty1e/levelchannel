# Style notes — что было переписано и почему

## Wave 4-mid.3 (17 июня 2026) — third pass: anglicisms cleanup

Главное: убрана англоязычная терминология и аббревиатуры из видимого текста статьи. Теперь весь контент звучит как российская публикация, а не калька с английского источника.

### Что меняли
- **CPD** (continuing professional development) → «профессиональное развитие»
- **ROI** (return on investment) → «окупаемость», «возврат на вложение»
- **RCT** (randomized controlled trial) → «рандомизированное контролируемое исследование»
- **ELA / Math / Sci** → «учителя гуманитарных предметов и естественных наук»
- **K-12** → «школа, классы с 1 по 11»
- **OECD** → ОЭСР
- **BLS** → Бюро статистики труда США
- **EPI / DfE / NEA** → расшифрованы при первом упоминании, дальше по-русски
- **adoption** → «освоение», «распространение»
- **weekly / monthly users** → «пользующиеся каждую неделю / каждый месяц»
- **active users / signups** → «активные пользователи / регистрации»
- **salary uplift / salary premium** → «прибавка к зарплате»
- **effect size / pooled effect / SD** → «размер эффекта / суммарный эффект / стандартное отклонение»
- **efficacy / effectiveness trial** → «контролируемое исследование на небольшой выборке / внедрение в обычные школы»
- **scale-up** → «масштабирование», «массовое внедрение»
- **vendor** → «поставщик», «платформа»
- **self-report / self-reported** → «самоотчёт респондентов»
- **micro-credentials** → «микро-сертификаты»
- **Master's degree** → «магистратура»
- **license renewal / leadership track** → «продление лицензии учителя / путь к административной позиции»
- **freemium** → «бесплатная база + платная подписка»
- **selection bias** → «смещение выборки»
- **L&D / execs** → «обучение и развитие сотрудников / руководители»
- **AI tool / AI literacy** → «ИИ-инструмент / ИИ-грамотность»
- **AI tutor** → «ИИ-репетитор»
- **peer review** → «независимая научная проверка»
- **any-use** → «хотя бы раз»

### Имена собственные оставлены
RAND, Pew, Gallup, Walton Family Foundation, EdWeek, Coursera, Khan Academy, MagicSchool, HolonIQ, LinkedIn, TALIS, AITSL, Eurydice, ISTE, NCTM, TESOL — при первом упоминании добавлены короткие русские пояснения.

### Что менялось в пайплайне
- `lib/llm/prompts/editorial_stylist.py` — добавлена таблица «Forbidden English jargon — переводим на русский» с 30+ запрещёнными терминами и их русскими эквивалентами.
- `lib/llm/prompts/research_editor.py` — добавлен идентичный блок «English jargon — переводим на русский», чтобы Editor выдавал русский текст ещё на стадии написания, не оставлял правки на Stylist.
