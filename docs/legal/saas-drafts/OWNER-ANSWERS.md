## Important deviation на #3 — только НПД, без ИП

Owner-уточнение: **ТОЛЬКО самозанятые (НПД), не ИП.**

Что это меняет:
- Регистрация на Operator-managed проверяет статус **НПД** через `npd.nalog.ru` API (НЕ общий ЕГРИП-статус).
- Учитель-ИП на Operator-managed → отклонение с сообщением "Operator-managed работает только с самозанятыми (НПД). Оформите НПД на nalog.ru — это бесплатно и занимает 5 минут".
- В оферте: п. 3.4 (или эквивалент) — "Тариф Operator-managed предоставляется только Заказчикам, состоящим в качестве плательщика налога на профессиональный доход (НПД, ст. 422-ФЗ)".

**Юридический смысл:** ИП и самозанятые отличаются налоговой моделью. Самозанятые проще — нет страховых взносов, упрощённый учёт, лимит дохода 2.4 млн ₽/год (что покрывает 99% маленьких учителей). Owner-выбор НПД-only делает Operator-managed строго для соло-учителей в эконом-сегменте.

**Что не меняется:** Free / Mid / Pro по-прежнему открыты для всех (включая физлиц без статуса, ИП, НПД, любую категорию). Ограничение НПД-only касается ТОЛЬКО Operator-managed.
