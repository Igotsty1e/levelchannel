8 концептов логотипа — LevelChannel

Без буквы L как иконки. Wordmark — "LevelChannel" с заглавной L. Тот же градиент бренда #C87878 → #E8A890. Первая четвёрка (A-D) — спокойные эссенциальные знаки. Вторая (E-H) — креативнее.

Первая четвёрка — эссенциальные знаки
Option A
Dot + wordmark
Точка-узел. Минимально, узнаётся быстро. Хорошо для favicon.
Option B
Concentric circle pulse
Концентрические круги = "канал/трансляция". Анимируется рябью при наведении.
Option C
Infinity ribbon
Непрерывная лента — "continuous learning". Скульптурная, узнаваемая форма.
Option D
Forward chevrons
Три шеврона ↗ = "level up", прогресс. Энергия движения.
Вторая четвёрка — более креативные
Option E
Equalizer bars
5 столбцов эквалайзера / "уровни сигнала". Прямая визуализация "level" + "channel" вместе. Анимируется bouncing-bars на hover.
Option F
Stair-steps monogram
Четыре поднимающихся ступени = рост учителя через тарифы (Free → Mid → Pro → Operator). Прямо отражает model "level up" в SaaS-смысле.
Option G
Orbit mark
Две пересекающиеся орбиты вокруг центрального узла = "учитель + ученик вращаются вокруг платформы". Современно, узнаваемо у Anthropic / Mistral / OpenAI стилистики.
Option H
Aperture blade
Грань диафрагмы / шестиугольный камень — "фокус, точность". Многоуровневая (три кольца = три тарифа). Хорош как favicon — сильный силуэт.
Скажи: A / B / C / D / E / F / G / H + scope
